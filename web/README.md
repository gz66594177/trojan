# StealthTrojan Web Admin Panel

React-based web admin panel for StealthTrojan.

## Features
- Device list and management
- Real-time monitoring via WebSocket
- Command center
- Data visualization
- Log management

## Setup
1. `npm install`
2. `npm start`

## API Backend
Connects to the backend server at `http://localhost:8000`

## API Key
Use the API key from the backend to authenticate requests.
